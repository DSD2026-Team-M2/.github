## About

DSD2026 Team M2 is the web-facing subteam of the broader **Limb Motion Recognition and Assistant** project.

We build and maintain the parts of the project that need to be **seen, explained, shared, and used on the web**:

- product-facing interfaces
- process and documentation portals
- recruitment and team communication pages
- readable demo and collaboration surfaces for delivery

This organization focuses on **M2-owned web deliverables** and related public-facing documentation, with an emphasis on clarity, maintainability, and collaboration.

## Working Style

We prefer:

- clear information architecture
- maintainable React + TypeScript code
- static-first deployment
- multilingual-ready content
- documentation-first collaboration
- calm, readable interfaces over unnecessary visual noise

## Repositories

### `project-main-web`

The main product-facing web application for the rehabilitation workflow.

It brings together **clinician**, **patient**, and **developer** entry points in one application, with charts, history views, and 3D visualization support.
